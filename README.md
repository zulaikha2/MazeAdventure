# Maze Text Adventure Game

This is a text-based maze adventure game developed for COE421L Software Engineering Lab.

## How to Run

Compile:
javac src/*.java

Run:
java -cp src Main

## Features
- Room navigation
- Inventory system
- Map and key mechanics
- NPC Wanderer with threading
- Command-based interaction

## Commands
- move north/south/east/west
- pick up map
- pick up key
- inventory
- look
- ask wanderer
- map
- quit
